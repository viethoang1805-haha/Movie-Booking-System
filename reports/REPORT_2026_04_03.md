### 1. Bùi Việt Hoàng

* Thiết kế kiến trúc hệ thống tổng thể
* Thiết kế Database gồm:

  * Users
  * Movies
  * Theaters
  * Showtimes
  * Seats
  * Bookings
  * Payments
  * Promotions
* Lựa chọn công nghệ:

  * Backend: NodeJS (NestJS)
  * Fontend: ReactJS
  * Database: MySQL 
  * Realtime: Socket.io
  * Authentication: JWT
* Đang thực hiện:

  * API Booking
  * Logic lock ghế (tránh đặt trùng)
  * Xử lý thanh toán cơ bản
